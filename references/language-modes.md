# Language Modes

## 必须先确认

每次调用 teacher skill 时，如果用户在本次请求中尚未明确指定语言，必须先询问：

> 教学 HTML 希望使用英文、中文，还是中英文对照？

等待用户回答后再读取和处理文档。不得根据用户界面语言、文档语言或历史偏好自行推断。

## 英文模式

- 全部教学内容使用英文。
- HTML 根元素使用 `<html lang="en">`。

## 中文模式

- 全部教学内容使用中文。
- HTML 根元素使用 `<html lang="zh-CN">`。

## 中英文对照模式

- 只生成一个 HTML 文件。
- HTML 根元素使用 `<html lang="zh-CN">`。
- 按最小内容单元紧邻配对，而不是按章节或大段落分区。

### 短内容：中英同行

标题、术语、短标签使用 `English 中文` 同行形式：

```html
<h2>
    Why Probability Matters
    <span lang="zh-CN">为什么要学习概率</span>
</h2>
<strong>
    Games
    <span lang="zh-CN">游戏</span>
</strong>
```

把标题、术语和短标签视为一个原子单元，英文与中文放在同一行或同一元素内。

### 长内容：英文后立即接中文

较长句子或段落使用英文在上、对应中文紧接下一行：

```html
<p class="bilingual-pair">
    <span lang="en">Probability is the mathematics of uncertainty.</span>
    <span lang="zh-CN">概率是研究不确定性的数学。</span>
</p>
```

长句、段落、列表项、表格单元、例题步骤和证明步骤都是独立原子单元；每个英文原子单元后必须立即出现对应中文。

## 禁止事项

- 禁止设置完整英文章节后再设置完整中文章节。
- 禁止使用 `ENGLISH` / `中文` 两个语言大板块。
- 禁止拆成两个 HTML 文件或两个页面。
- 数学公式若两种语言共用且完全相同，只展示一次；公式前后的文字解释仍按原子单元紧邻配对。
- 不要使用上边框、大间距或语言标签把两种语言视觉上分成大板块，也不要拆成左右栏。
